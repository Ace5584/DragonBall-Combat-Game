# DragonBall-Combat-Game

## Controls:
## Player 1:
  - W -> Jump
  - S -> Teleport Downwards
  - D -> Right
  - A -> Left
  - Q -> Shoot
  - R -> Punch
  - F -> Teleport to The Cornor
## Play 2:
  - Arrow Up -> Jump
  - Arrow Down -> Teleport Downwards
  - Arrow Right -> Right
  - Arrow Left -> Left
  - Space -> Shoot
  - N -> Punch
  - B -> Teleport to The Cornor
